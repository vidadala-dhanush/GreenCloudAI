# GreenCloudAI

Lightweight project for analyzing cloud usage and energy efficiency.

## Structure
- `backend/` — core processing and model code
- `frontend/` — minimal UI or demo app
- `dataset/` — sample data files

## Setup
Install dependencies:

```bash
python -m pip install -r requirements.txt
```

Run the frontend (example):

```bash
python frontend/app.py
```

Run the backend (example):

```bash
python backend/greencloud_ai.py
```

## Notes
- Add or update `.gitignore` to avoid committing virtual environments and caches.
